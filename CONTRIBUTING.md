# Contributing to Awesome Android UI Fun

Thank you for your interest in contributing! This list focuses on **high-quality**, **open-source**, and **privacy-respecting** Android apps with a fun or unique UI.

## Rules

### Rule 1: Open Source Only
All apps must be released under an [Open Source Initiative (OSI) approved license](https://opensource.org/licenses/). Proprietary or "source available" apps will be rejected.

### Rule 2: Privacy First
Apps must not contain trackers or advertisements. We prioritize privacy and user respect.

## How to Add an App

To maintain the grid layout, please use the following HTML template when adding a new app. Do not break the table structure!

### Template
Copy and paste this code into the appropriate category table in `README.md`:

```html
<td>
  <div align="center">
    <img src="https://ui-avatars.com/api/?name=App" width="64" alt="Icon" />
    <h3>App Name</h3>
    <code>[Tag]</code>
    <br/>
    <p>A witty, 1-sentence summary of the app.</p>
    <a href="URL_TO_SOURCE">
        <img src="https://img.shields.io/badge/Get%20it-F--Droid-3DDC84?style=for-the-badge&logo=android&logoColor=white" alt="Get it on F-Droid">
    </a>
  </div>
</td>
```
